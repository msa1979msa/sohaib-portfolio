# Muhammad Sohaib Ahmed — Portfolio v2

AI Engineer | Backend Systems (FastAPI) | Multi-Agent Architect | GPT Applications

## Tech Stack
- Next.js 14 (App Router) + TypeScript
- Tailwind CSS + Custom CSS Variables
- Lucide React Icons
- Syne + DM Sans + JetBrains Mono fonts
- Vercel deployment

## Pages
- `/`                      → Home (Hero, Arch Diagram, Services, Projects, Stack, Certs)
- `/projects`              → All 4 projects
- `/projects/healthcare`   → AI Healthcare Management Platform (NEW)
- `/projects/erp`          → Extreme MSA ERP System
- `/projects/powerbi`      → Power BI Business Analytics
- `/projects/mobile`       → Cross-Platform Mobile Apps
- `/about`                 → Bio, Skills, Experience, Education, Certifications
- `/contact`               → Contact Form + Social Links

## Quick Start
  npm install
  npm run dev
  # Visit http://localhost:3000

## Add Resume
  Place resume.pdf in the /public folder

## Deploy
